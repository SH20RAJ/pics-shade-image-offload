== Description ==
Offload media from WordPress to Pics Shade - Image Hosting Made Easy.

== Installation ==
1. Upload the `pics-shade-image-offload` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Settings' -> 'Pics Shade Image Offload' to configure the plugin.
