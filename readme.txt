== Description ==
Pics Shade Media Offload + Optimize + Resize is a plugin designed to enhance your WordPress media management. It offloads your media files to Pics Shade, optimizes images for faster delivery, and includes resizing capabilities to improve site performance.

== Benefits ==
* **Offload Images**: Automatically transfer your WordPress media to Pics Shade, reducing server load and improving performance.
* **Optimize Links**: Utilize optimized CDN links for faster image delivery.
* **Manage Settings**: Easily configure API keys, default tags, and more.
* **Automatic Deletion**: Optionally delete local images after offloading.

== Installation ==
1. **Download the Plugin**:
   * Visit the [GitHub repository](https://github.com) and click on "Download" or use the [Direct Download link](https://github.com).

2. **Upload to WordPress Dashboard**:
   * Log in to your WordPress dashboard.
   * Navigate to 'Plugins' > 'Add New'.
   * Click on 'Upload Plugin' and choose the `pics-shade-image-offload.zip` file.
   * Activate the plugin through the 'Plugins' menu in WordPress.

3. **Configure Settings**:
   * Once activated, configure your settings under 'Settings' > 'Pics Shade Image Offload'.

== Frequently Asked Questions ==
= How do I get my Pics Shade API key? =
Visit the [Get API Key](https://docs.pics.shade.cool/api-reference/get-api-key) section on the Pics Shade documentation.

= Can I delete local images after offloading? =
Yes, the plugin provides an option to automatically delete local images after they have been successfully offloaded to Pics Shade.
